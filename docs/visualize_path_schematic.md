# KiCad Path visualisation Guide schematic

## Overview

The schematic Path visualisation Method allows you to:

1. Mark a specific Path through the connected wires in your schematic file
2. to visualize those paths the schematic is plotted in a svg file by claude
3. See how components are connected through your schematic and over hierarchical labels over multiple sheets
4. Paths are being overwritten by adding new ones to the files

## Using Path Visualisation Features

### Highlight Paths 

```
Can you highlight The Path in schematic from U1 to R2?
```

This will provide:
- colored lines in the svg file
- visual and interactive of the svg with the path in your claude response
- you can choose your own color
